# 记忆管理工具 (Memory Manager)

此技能提供了一个简化的接口，让 Alice 能够轻松地维护其短期记忆，而无需手动处理文件追加、日期标题或时间戳。

## 核心功能

- **自动化日期管理**: 自动检测当天日期并添加 `## YYYY-MM-DD` 标题（如果不存在）。
- **自动时间戳**: 为每条记录自动添加 `[HH:MM]` 前缀。
- **稳健追加**: 确保记录以正确的 Markdown 列表格式追加到文件末尾。

## 使用方法

在终端运行：

```bash
python skills/memory_manager/memory_tool.py "**事件**: 用户咨询项目架构 | **行动**: 详细介绍了 agent.py 和 skills 系统"
```

## 推荐时机

- 完成了一个复杂的调研任务后。
- 捕获到了用户的重要偏好或新要求时。
- 达成了一个项目的里程碑或做出了重要决策时。

## 记录格式建议

虽然工具会自动添加时间戳，但建议按照以下格式组织内容：
`**事件**: <发生了什么> | **行动**: <你做了什么>`
